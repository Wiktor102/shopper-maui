# ShopperMaui

**Polski**

ShopperMaui to prosta aplikacja do tworzenia list zakupów, stworzona jako projekt szkolny. Jej celem jest pokazanie podstawowych umiejętności programistycznych z wykorzystaniem .NET MAUI. Projekt nie jest przeznaczony do dalszego rozwoju ani do użytku produkcyjnego.

Jeśli szukasz bardziej rozbudowanej aplikacji do list zakupów, zapraszam do mojego innego projektu: [Shopper](https://www.wiktorgolicz.pl/shopper/)

---

**English**

ShopperMaui is a simple shopping list application created as a school project. Its purpose is to demonstrate basic application development skills using .NET MAUI. This project is not intended for further development or production use.

If you are interested in a fully-featured shopping list app, please visit my other project: [Shopper](https://www.wiktorgolicz.pl/shopper/)
